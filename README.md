# 📊 Customer Churn Prediction Dashboard

An interactive Machine Learning dashboard built using Streamlit
to predict customer churn.

---

## 🚀 Features

- Real-time churn prediction
- Multiple ML models
- Model comparison
- Feature importance visualization
- Confusion matrix
- Classification report
- Interactive UI

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- XGBoost
- LightGBM
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## ▶️ Run Locally

### 1. Clone Repository

```bash
git clone <your-github-link>
```

### 2. Install Requirements

```bash
pip install -r requirements.txt
```

### 3. Run App

```bash
streamlit run app.py
```
